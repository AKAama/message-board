# 新年祝福墙

一个简单的新年祝福留言板应用，允许用户发送和查看新年祝福。

## 功能特点

- 发送新年祝福
- 查看所有祝福
- 精选祝福展示
- 管理员模式
- 响应式设计

## 技术栈

### 前端
- HTML5
- CSS3
- JavaScript (原生)

### 后端
- Node.js
- Express
- MySQL
- Sequelize ORM
